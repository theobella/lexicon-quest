# Lexicon Quest - Walkthrough

## Overview
Lexicon Quest is a fast-paced word definition game designed for word lovers. Players must identify words from their definitions under time pressure.

## Features
- **Three Difficulty Levels**:
  - **Easy**: 3 Options (Distinct Words), 30 seconds.
  - **Medium**: 3 Options, Faster timer (20s).
  - **Hard**: 3 Options (Similar Words), 15s.
- **Scoring**: Points based on difficulty + Speed Bonus.
- **Persistence**: High scores are saved to your browser.
- **Social**: Share your score with friends.

## How to Play
1. **Choose Difficulty**: Select Easy, Medium, or Hard from the home screen.
2. **Answer Quickly**:
   - Read the definition.
   - Click the correct word.
   - Keep an eye on the timer!
3. **Build a Streak**: Consecutive correct answers boost your streak (visual only for now).
4. **Game Over**: If you run out of time or guess wrong, the game ends.
5. **Share**: Click "Share Result" to challenge friends.

## Technical Details
- **Framework**: Next.js 14 (App Router)
- **Styling**: CSS Modules with custom "Academic/Retro" theme.
- **State**: React Context + LocalStorage.
